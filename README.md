# Microwave Oven simulation using Picsimlab with PIC16F877A microcontroller

## Overview
This project was developed as part of my **Embedded Systems Internship (Remote, 1 Month)** at **Emertxe Information Technologies, Bangalore**.  
It simulates the functionality of a **Microwave Oven** using **PICSimLab** with the **PIC16F877A microcontroller**, programmed in **Embedded C** with **MPLab IDE v5.35**.

## Features
 **Cooking Modes:** Micro, Grill, Convection, Start (default 30s)
 
 **User Interface Screens:**
    Power ON screen,
    Mode selection,
    Set Time / Set Temperature and
    Display Remaining Time
    
 **Functional Controls:**
    Time entry (MM:SS)
    Temperature setting (for convection mode)
    Pre-heating sequence
    Pause, Resume, Stop options
    
 **Peripheral Interfacing:**
    LEDs, switches, matrix keypad, timers, CLCD
    Fan simulation

---

## Tools & Technologies
- **Programming Language:** Embedded C  
- **IDE:** MPLab IDE v5.35  
- **Simulator:** PICSimLab  
- **Microcontroller:** PIC16F877A
- **Board used:** PICGenios

---

## Outcomes
- Hands-on experience with **microcontroller architecture, interrupts, polling, and timers**  
- Debugging and simulation of real-time embedded systems  
- Designing user interfaces for embedded applications  
- Implementing realistic appliance functionality in a simulated environment
